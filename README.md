# 宅建学習サイト

宅建試験の学習用に、分野別の教材スライドと一問一答をまとめる静的HTMLサイトです。

## 構成

- `index.html`: トップページ
- `seigen/`: 法令上の制限
- `seigen/kenchiku.html`: 建築基準法の学習ページ
- `kenri/`: 権利関係
- `gyouhou/`: 宅建業法
- `images/kenchiku/`: 建築基準法ページで使用するスライド画像

## 現在の教材

- 法令上の制限
  - 建築基準法

## 使い方

`index.html` をブラウザで開くとトップページを表示できます。

建築基準法ページでは、スライド画像を番号順に表示し、下部の一問一答で復習できます。

## メモ

このサイトは外部ライブラリを使わない静的HTMLとして作成しています。
